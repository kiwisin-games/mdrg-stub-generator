## Lua asset loading pattern

Lua references the JSON, not the PNG directly:

```lua
local packed = ModUtilities.GetPackedTexture(CurrentModGuid, "/Folder/file.json")
itemprefab.AddTexture(packed)
```

The JSON `TextureName` points to a PNG in the same folder.
